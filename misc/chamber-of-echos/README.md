# 題材とする脆弱性

コバートチャネル攻撃  
※ 本来通信に使われることを想定していない手段を使って情報を密かに外部へ送信する (脆弱性というよりはプロトコルの仕様)


# 実現するためのテーマ

ネットワーク層での隠された通信チャネルの利用


# 想定する参加者が解答までに至る思考経路

1. 与えられたホストに対して ICMP 通信 (ping) を試す
1. 異常な ICMP Echo Reply を検出
1. ICMP Echo Reply のペイロードに意味のあるデータがあることに気づく
1. 何度か ICMP Echo Request を送り、パケットを収集する
1. ヒントで与えられた鍵を利用し、収集したペイロードを復号する
1. 復号したペイロードの先頭にインデックスが振られているのに気づく
1. インデックスに従って並べ替え、結合して提出する


# 想定する難易度

Hard
