# 題材とする脆弱性

ディレクトリ・トラバーサル

# 実現するためのテーマ

ディレクトリ・トラバーサルの脆弱性を持つウェブサーバアプリケーションがある.  
proc ファイルシステムから実行時の引数情報にアクセスするとフラグ入手できる.

# 想定する参加者が解答までに至る思考経路

1. クエリ文字列から任意のファイルを指定して閲覧できることに気づく
1. ヒントとなるログファイルから、 procfs にアクセスできることに気づく
1. cmdline にて実行時引数に渡されている flag を入手できる
