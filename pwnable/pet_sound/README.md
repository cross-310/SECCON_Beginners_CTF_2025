# 想定する参加者が解答までに至る思考経路

定義されている構造体には、文字列と関数ポインタを持っている。
入力箇所は一回であるが、文字列サイズを超えているのでオーバーフローが可能。
確保しているヒープ領域の様子を出力しているので、それを基に入力位置と書き換えられる関数アドレスの位置を見て、オーバーフローさせて、書き換える。
