# 題材とする脆弱性

0x10バイトだけのオーバーフロー

# 実現するためのテーマ

elf baseやlibc baseのリークをできるようになる。
vuln関数の途中へjmpするという発想を持てるようにする。
one_gadgetを使えるようになる。

# 想定する参加者が解答までに至る思考経路

read関数で0x10バイトのオーバーフローがあることに気がつく。
何らかのアドレスをリークしてret2vulnしたいが、ASLRが有効であるので、下1バイトだけ書き換える。このとき、elf baseを入手できる。
また、レジスタの値を確認するとrdiがlibcのアドレスを指していることがわかる。vulnのputs関数に飛ばすと、libc leakをすることができる。
libc baseを得られたので、one_gadgetを使う。
