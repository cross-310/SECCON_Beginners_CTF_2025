# Math Myth

RSA暗号の問題です。特殊な構造を持つ素数を利用した脆弱性を突きます。

## 問題概要

通常のRSAパラメータ（n, e, c）に加えて、追加の値 r が与えられています。

素数 p は以下の特殊な形をしています：
```
p = q² * A + g(q, salt) * r
```
ここで：
- A は r の次の素数
- g(q, salt) はSHA-512を使った擬似ランダム関数（512ビット出力）

## 解法

1. α = nextPrime(r) - r を計算
2. a = n * α⁻¹ mod r = q³ mod r を求める
3. r を素因数分解（56ビット素数4個の積）
4. 各素因数上で3乗剰余根を計算
5. 中国剰余定理（CRT）で q ≡ Q (mod r) を合成
6. q = Q + k*r の形で真の q を探索（約10ビットの探索が必要）
7. 通常のRSA復号でフラグを取得

## パラメータ設計

- BITS_q = 280
- BITS_r = 4 * 56 = 224
- g(q, salt) は SHA-512 で 512ビット出力
- betaパラメータは不要（SHA-512の出力で十分なランダム性）
